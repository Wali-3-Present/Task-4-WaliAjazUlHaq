##Project Objective
This repository hosts **Project 4 (Data Visualization)** for the DecodeLabs Industrial Internship. Moving past programmatic calculations, this phase builds high-definition reporting plots designed to map corporate milestones and logistical trends clearly.

###Visual Architecture Milestones Delivered:
**Pareto Revenue Profiles:** Constructed horizontal categorical bar segments to rank exactly which item variants dominate consumer expenditure markets.
**Temporal Sales Trajectories:** Modeled ongoing transactional revenue changes across running timelines to spot market seasonality patterns.
**Operational Pipeline Slices:** Rendered structural order distribution percentages to quickly pinpoint fulfillment friction points or churn values.
